Gesture Plugin

O Gesture Plugin é uma ferramenta inovadora que permite aos utilizadores utilizarem gestos para interagir com o sistema de forma intuitiva e eficiente. Com este plugin, é possível controlar aplicações, navegar por menus e executar ações com simples movimentos das mãos. Desenvolvido com tecnologia avançada, o Gesture Plugin transforma a forma como os utilizadores interagem com a tecnologia, proporcionando uma nova dimensão de conveniência e inovação. Foi desenvolvida para a cadeira Projeto4 da universidade IPVC-ESTG da licenciatura de Engenharia Informática, com o orientador Professor Pedro Moreira e os alunos Diogo Gaspar e Gustavo Barbosa.

Sobre os Desenvolvedores

O Gesture Plugin foi desenvolvido por uma equipa dedicada de programadores apaixonados por tecnologia e inovação. Com o objetivo de criar uma experiência de utilizador única, a equipa trabalhou arduamente para integrar gestos intuitivos no plugin, tornando-o acessível e fácil de usar. Acreditamos que a tecnologia deve ser simples e eficaz, e o Gesture Plugin é um reflexo desse princípio.
